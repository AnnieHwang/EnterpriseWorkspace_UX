Introduction
---------------------------------------------
권장설치 버젼은 다음과 같습니다.

You will need:
    * CentOS 6.x or Higher
    * JDK 1.8 (recommended Oracle JDK)
    * MySQL 5.5 or Higher
    * Apache Zookeeper 3.4.10 or Higher
    * Need to Ansible Installation on distribution server
    * Hadoop Cluster (2.7.3 recommended)


싱글과 분산환경의 차이
분산이기 때문에 파일 시스템이 있어함

    * 하둡을 설치해야 함
    * 하둡은 Metatron Discovery의 StagingDB 사용을 위해서 필요함
