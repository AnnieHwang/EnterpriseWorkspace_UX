Version¶

우리의 대표 2.7.3로 기준으로 default로 제공함 (암바리 기준으로 2.6.x임) 3점대도 지원함. 이 경우 따로 building을 해야함 (현재 3.1.4 HDP 테스트 했음) 버젼을 사이트 별로 따로 관리되어야 함 (현재 hadoop 2.7대로 druid가 building 되어 있음. hadoop 3.x 대 버젼으로 사용하려면 druid를 building해야 함) 우리는 어떤 버젼을 보장합니다. hadoop minor버젼에 영향을 많이 받기 때문에..위의 버젼이 아닌 경우, 테스트 해보기 바란다.
